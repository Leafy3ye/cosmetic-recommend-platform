# 化妆品推荐平台 - 前端项目

## 项目简介
基于Vue3的化妆品推荐平台前端系统，采用现代化的UI设计，提供用户友好的购物体验。

## 技术栈
- **框架**: Vue 3.3
- **构建工具**: Vite 4.4
- **路由**: Vue Router 4.2
- **状态管理**: Pinia 2.1
- **UI组件**: Element Plus 2.4
- **HTTP客户端**: Axios 1.5
- **图表库**: ECharts 5.4

## 项目结构
```
frontend/
├── src/
│   ├── api/                # API接口
│   │   ├── user.js        # 用户接口
│   │   ├── product.js     # 商品接口
│   │   ├── cart.js        # 购物车接口
│   │   └── order.js       # 订单接口
│   ├── components/         # 公共组件
│   │   └── ProductCard.vue # 商品卡片
│   ├── views/             # 页面组件
│   │   ├── Login.vue      # 登录注册页
│   │   ├── Home.vue       # 首页
│   │   ├── Products.vue   # 商品列表
│   │   ├── ProductDetail.vue # 商品详情
│   │   ├── Cart.vue       # 购物车
│   │   ├── Orders.vue     # 订单列表
│   │   └── Profile.vue    # 个人中心
│   ├── router/            # 路由配置
│   ├── stores/            # 状态管理
│   ├── utils/             # 工具函数
│   │   └── request.js     # axios封装
│   ├── App.vue            # 根组件
│   └── main.js            # 入口文件
├── index.html             # HTML模板
├── vite.config.js         # Vite配置
└── package.json           # 项目依赖
```

## 快速开始

### 环境要求
- Node.js 16+
- npm 或 yarn

### 安装依赖
```bash
cd frontend
npm install
```

### 开发模式
```bash
npm run dev
```
访问地址: http://localhost:5173

### 生产构建
```bash
npm run build
```

### 预览构建结果
```bash
npm run preview
```

## 接口配置
后端API地址配置在 `src/utils/request.js` 中：
```javascript
baseURL: 'http://localhost:8080/api'
```
